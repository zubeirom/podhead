Software Developement Lifecycle:

Design:
Lean on Google Podcast Design or Vercel (formerly Zeit)

Database:
MongoDB and Atlas as hosting service

Search Engine:
Use elasticsearch as full text search engine

API:
Use Nest.js with Typescript

Blob Storage:
Google Cloud Storage

Client routes


/channels = List of channels

/new = Add new Channel

/${channelusername} = Channel

/${channelusername}/new = New Episode

/${channelusername}/feed.xml = xml path

/${channelusername}/epid = Episode - edit or delete

Design highly inspired by google Podcast

     Channel Model:
         channelImage: string,
         channelName: string,

         episodes: Episode[],
         channelHomePage: string,
         feedUrl: string,
         category: Category,
         createdAt: Date,
         updatedAt: Date,

     Episode Model:
         title: string,
         description: string,
         length: string,
         audioURL: string,
